# 60-Second Demo Checklist

1. Open Home
- Show hero, feature cards, and promo banner
- Highlight store availability by location

2. Change Location
- Use the location selector in the header
- Show how stores swap between “Available” and “Unavailable”

3. Open a Store
- Click a store card to navigate to `/stores/[slug]`
- Show store header, info boxes, and featured products

4. Browse and Filter
- Use category chips to filter products
- Try search with `/` shortcut
- Toggle grid/list and sort

5. Add to Cart
- Click “Add” then use the stepper
- Open the cart with `c`

6. Checkout
- Click “Proceed to Checkout”
- Show form + order summary
- Enter a phone number to enable “Place Order”
